The Behavior Test

Because guessing the behavior of an application becomes as complicated as us, we wanted to create a simple tool allowing you to write your behavior scenarios on your application in order to be able to launch them automatically backwards.

Indeed, we have created a Text Editor that allows you to write HTML files. These HTML files are then run on different browsers like Chrome and / or Firefox.

You have the possibility of customizing all of these files that we have called Tastee File. These files are then integrated into a test report once their executions are complete.

You also have the possibility to create your own functions but also to use keywords or variables to generalize all of your files.
